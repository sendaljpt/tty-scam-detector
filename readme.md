# 🚨 Scam Detector CLI

Tool CLI sederhana untuk mendeteksi potensi **website scam / phishing** berdasarkan berbagai indikator seperti domain, SSL, DNS, dan pola URL.

Dibuat untuk edukasi cyber security & content penetration testing 🔍

---

## ✨ Features

- ✅ Domain age & expiration check (WHOIS)
- ✅ Suspicious keyword detection (brand abuse)
- ✅ Fake government domain detection (`go.id`)
- ✅ DNS resolution check
- ✅ HTTPS & SSL validation
- ✅ SSL issuer info
- ✅ Redirect analysis
- ✅ Subdomain detection
- ✅ MX record (email server) check
- ✅ IP address lookup
- ✅ External links analysis
- ✅ URL obfuscation detection
- ✅ Port check
- ✅ Nameserver check
- ✅ Typosquatting detection (domain mirip)
- 🎯 Scoring system + risk classification
- 📊 Output table (rapih & readable)
- 🎨 Colored CLI output
- ⚡ Progress bar (tqdm)
